Tables are used to display data in rows and columns. They are very useful for organizing information such as schedules, results, or comparisons.

Key Table Tags:

  • table: — Defines the table.
  • tr: — Defines a row.
  • th: — Defines a table header (bold and centered by default).
  • td: — Defines a table cell (data).

Example Table:

Student Name Subject Marks
Sindhu HTML 95
Ravi CSS 88
Anita JavaScript 92

Example Uses:

  • Student marks or attendance records.
  • Product price lists.
  • Timetables and schedules.

Tables make it easy to view data in a structured and professional layout.
