What is HTML?

HTML consists of a series of Elements.

  1. HTML:Root element of an HTML Page.
  2. Head This element contains meta information about the HTML Page.
  3. TitleThis element specifies a title for the HTML Page.
  4. h1 This element defines a large heading
  5. pThis element defines a paragraph.
