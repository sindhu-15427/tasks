CSS Selectors

Explanation of Selectors

  • Element Selector – selects elements by name (p, h1, div)
  • Class Selector (.) – selects elements with class name
  • ID Selector (#) – selects an element with a unique ID
  • Group Selector (,) – applies same style to multiple tags
  • Universal Selector (*) – selects all elements
  • Attribute Selector – selects elements with specific attributes
