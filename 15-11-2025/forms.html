Forms in HTML

Explanation

HTML forms are used to collect user input. They contain form controls like text boxes, radio buttons, checkboxes, dropdowns and submit buttons.

Common Form Elements:

  • form: – wrapper for all input controls
  • input: – text, email, password, radio, checkbox
  • select: – dropdown list
  • textarea: – multi-line input
  • button: – submit button
