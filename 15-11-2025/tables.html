HTML Tables

HTML tables allow web developers to arrange data into rows and columns.

Table Explanation

table: – Creates a table

tr: – Table row

th: – Table heading (bold + centered)

td: – Table data/cell

Tables are used to display data in rows and columns.

Table Example
