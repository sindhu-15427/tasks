What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS is one of the core technologies of the World Wide Web, alongside HTML and JavaScript. It controls the visual appearance of web pages, including colors, fonts, layouts, and responsive behaviors.

CSS Syntax

A CSS rule consists of a selector and a declaration block:

Syntax:

selector { property: value; property: value; }

Example:

p { color: blue; font-size: 18px; }

CSS Selectors

  • Element Selector — Select elements based on their tag name.
  • Class Selector — Select elements based on their class attributes.
  • ID Selector — Select elements based on their id attributes.
  • Universal Selector — Select all elements(*) in the document.
  • Highlight Selector (::selection) — changes color when text is selected
  • Descendant Selector — Selects element that are descendants of another element(space between selectors).
  • Child Selector — Selects elements that are direct children of another element(use > symbol).
  • Adjacent Sibling Selector —Selects an element that is directly after another specific element(use + symbol).
  • Attribute Selector — Selects elements with a specific attribute.
  • Pseudo-Class Selector — Selects elements when you hover over them.
